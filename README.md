# WhatsApp Reaction Hub

Website untuk mengirim reaction ke chat WhatsApp dengan emoji kustom.

## Cara Pakai:
1. Scan QR code dengan WhatsApp
2. Masukkan URL chat WhatsApp
3. Masukkan emoji (pisah dengan koma)
4. Masukkan jumlah reaction
5. Klik "Kirim Reaction"

## URL Contoh:
- Chat pribadi: `https://wa.me/6281234567890`
- Grup: `https://chat.whatsapp.com/ABC123def456`
